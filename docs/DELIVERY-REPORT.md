# 🚀 对标鲸商城PRO 深度迭代(2026-06-22,基于 fkdemo 实测)

> 70 代理工作流实测对标(`docs/jf-benchmark.md` / `iteration-plan.md`)后,按 TDD/守绿/一任务一提交逐项补齐。**从零重建迁移 + 387 后端测试全绿,前端构建通过。**

## ① 商品类型四分法(地基,业主已批)
- `goods_type`:1卡密 / 2知识 / 3资源 / 4权益,与发货方式 `type`(自动/手动)正交;存量商品=卡密,向后兼容。
- **下单/发货按类型路由**(关键路径 TDD):非卡密类跳过 cards 行锁预占、发 `delivery_message` 内容并结算;卡密类一卡一售/并发/验签**完全不变**。
- 店铺**按类型分组陈列**(⚡卡密/☘️知识/💎资源/👑权益);买家端类型徽标 + 发货物按类型称谓;商户表单类型选择器;库存语义按类型(非卡密「现货」)。

## ② 营销
- **优惠券**(关键路径 TDD):满减/折扣,门槛/封顶/防0元/有效期/库存;商户 CRUD 页;买家结算填码试算+抵扣;下单核销、结算/支付按应付、**退款反核销**。
- **限时折扣**(关键路径 TDD):商品价格层时段促销,窗口内改单价;买家「限时」徽标+原价划线;商户表单设折扣价+窗口。

## ③ 售后资金闭环(关键路径 TDD)
- **退款**:卡密回库 + 反向资金(精确依据实际结算流水冲收入+佣金回冲)+ 优惠券反核销;平台后台一键退款。
- **投诉全链路**:买家发起→商户回复→买家申请介入→平台仲裁(可联动退款);买家查单页入口 + 商户处理页 + 平台仲裁页。

## ④ 门户内容
- `articles` 资讯/常见问题/单页:后台 CRUD + 门户公开 `/index/articles` API + 买家资讯/FAQ 页(分类筛选)+ 顶栏门户入口。

## 总后台模块(对标 fkdemo 菜单)
- 平台/商户**利润指标**(今日/昨日/本月/累计)· 仪表盘**待处理接入投诉计数**
- **买家黑名单**(下单前拦截)· **操作日志/审计**(退款/投诉/黑名单/商户/提现全埋点 + 查看页)
- 订单查询密码 · 分类图 · 商户毛利
