## ✅ 已修复并复验(本 session)

### HIGH(影响可用性)
- [x] **商户审核/邀请码/内容管理 操作列按钮被裁成 1px** — 根因 `DataTable` td 对所有列套 `overflow:hidden+maxWidth` 撞 ops 列 `width:1`。修:DataTable 加 `nowrap` 列支持,ops 列标 `nowrap`。截图复验按钮完整可点。
- [x] **admin 商品(跨商户)用不存在的 `--color-*` 变量** — 致次要文字变黑、零库存红警失效、`icon=` 图标不渲染。修:全部改真实 token + `iconLeft`。复验零库存红警回来了。
- [x] **分类管理 分类图破图框** — `<img>` 无 onError。修:改 background-cover div,失败显干净灰底。
- [x] **内容管理 缺资讯/FAQ/单页(后台不完整)** — 后端早有 `/admin/articles` CRUD,前端没接。修:加 4 Tab + 文章增删改 + API 方法。**端到端实测创建写入成功。**
- [x] **详情底部购买条留白不足** — 132→176px+安全区(注:fullPage 截图的"重叠"系固定元素伪影,视口态实测无遮挡)。

### MED
