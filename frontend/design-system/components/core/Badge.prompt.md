Status pill — stock, order state, trust markers.

```jsx
<Badge variant="success" dot>有货</Badge>
<Badge variant="danger" dot>缺货</Badge>
<Badge variant="secure" icon={shieldIcon}>担保交易</Badge>
<Badge variant="brand" solid>热卖</Badge>
```

Variants: `neutral | brand | success | pending | danger | secure`. `solid` for filled high-emphasis, `dot` for a status dot, `icon` for a leading glyph.
